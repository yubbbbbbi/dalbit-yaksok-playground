# 달빛약속 (한글 컴퓨터 언어) 인터프리터

한글 컴퓨터 언어인 **'달빛약속'**을 수업에 활용하기 위해, 인터프리터 기능만 별도로 분리하여 구축한 독립 배포 저장소입니다.

## 프로젝트 소개
이 프로젝트는 교육용 한글 언어인 '달빛약속'의 핵심 기능을 웹에서 바로 실습할 수 있도록 인터프리터 부분만 별도로 분리하여 제작되었습니다.

* **원본 프로젝트**: [달빛약속 공식 사이트](https://dalbit-yaksok.postica.app/)
* **실습 사이트**: [달빛약속 플레이그라운드 바로가기](https://yubbbbbbi.github.io/dalbit-yaksok-playground/)
